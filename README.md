# delphi-indy-rest-template
Delphi starter template for client–server applications using Indy and REST API.

## Purpose

This repository provides a basic and reusable foundation for
Delphi client–server projects based on:

- Indy HTTP Server
- RESTful API
- JSON data exchange

The project is intended to be used as a starting point for
custom applications and further extensions.

## Architecture

- **Server**
  - Indy HTTP Server
  - REST API endpoints
  - JSON request/response
- **Client**
  - Delphi VCL application
  - REST communication with the server

## Project Structure
/server
/src
/client
/src
/shared
models.pas

## Getting Started

1. Open the server project file (`server.dproj`)
2. Build and run the server
3. Open the client project file (`client.dproj`)
4. Run the client and test API communication

## Requirements

- Delphi 12.x or newer
- Indy (bundled with Delphi)
- Windows OS

## Notes

This is a template project, not a finished product.
It is designed to be modified, extended, and adapted
to specific project requirements.
